# T-DEV-810 — ZoidBerg (FU1)

Objectif du FU1 : mettre en place un pipeline minimal et reproductible :
1) importation/organisation du dataset
2) exploration + statistiques de base
3) preprocessing minimal (format/normalisation)
4) premier modèle baseline + premières métriques

## Pré-requis
- Python 3.13+

## Installation
Cloner le repo, puis à la racine :
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
